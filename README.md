What is not working:<br>

<h3>Setting up Client<h3>
  <ul>
    <li>cd client</li>
    <li>npm install</li>
    <li>npm start</li>
  </ul>
  
<h3>Setting up Server<h3>
  <ul>
    <li>cd server</li>
    <li>npm install</li>
    <li>Create .env file</li>
    <li>add SECRET and DB_STRING</li>
    <li>npm start</li>
  </ul>
